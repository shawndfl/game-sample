# game-sample

This is a sample game with opengl es graphics

#Setup

Ubuntu

```Bash
sudo apt install libglfw3-dev libpng-dev
```

Fedora

```Bash
sudo dnf install glfw-devel libpng-devel mesa-libGLES-devel
```

# Build 

```
make -j 10
```

# Run 

```
./BlockSimulatedKingdom
```
